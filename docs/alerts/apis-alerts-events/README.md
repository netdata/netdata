# 9. APIs for Alerts and Events

Netdata provides APIs for querying and managing alerts programmatically.

## What You'll Find in This Chapter

| Section | API Endpoint | Purpose |
|---------|--------------|---------|
| **9.1 Query Current Alerts** | `/api/v1/alarms` | List active alerts |
| **9.2 Alert History** | `/api/v1/alarm_log` | Get transitions |
| **9.3 Inspect Variables** | `/api/v1/alarm_variables` | Debug expressions |
| **9.4 Health Management** | `/api/v1/health` | Enable/disable alerts |
| **9.5 Cloud Events API** | Cloud API | Query events from Cloud |
