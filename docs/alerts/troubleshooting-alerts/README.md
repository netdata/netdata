# 7. Troubleshooting Alert Behaviour

This chapter provides systematic approaches to diagnose and fix alerts that don't behave as expected.

## What You'll Find in This Chapter

| Section | Problem |
|---------|----------|
| **7.1 Alert Never Triggers** | Alert should fire but stays CLEAR |
| **7.2 Always Critical/Warning** | Alert stuck in non-CLEAR state |
| **7.3 Alert Flapping** | Rapid status changes |
| **7.4 Variables Not Found** | Expression errors |
| **7.5 Notifications Not Sent** | Alert fires but no notification |

## How to Navigate This Chapter

- Start with **7.1** if alerts never fire
- Jump to **7.2** if alerts stay critical
- Use **7.3** for flapping issues
- Go to **7.4** for variable errors
- Check **7.5** if notifications don't arrive

## What's Next

- **7.1 Alert Never Triggers** covers diagnostic steps
- **7.2 Always Critical** for calculation errors
