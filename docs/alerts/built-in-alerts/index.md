# 11. Built-In Alerts Reference

This chapter documents the stock alerts that Netdata includes out of the box. These alerts cover common infrastructure scenarios and serve as templates for custom alerting.

## What You'll Find in This Chapter

| Section | What It Covers |
|---------|----------------|
| **11.1 System Resource Alerts** | CPU, memory, disk, I/O, and network interfaces |
| **11.2 Hardware Alerts** | Disk SMART, thermal sensors, and hardware health |
| **11.3 Application Alerts** | Databases, web servers, caches, and message queues |
| **11.4 Network Alerts** | Network interface errors, drops, and speed |
| **11.5 Container Alerts** | Docker and Kubernetes container metrics |
