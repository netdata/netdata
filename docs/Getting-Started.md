## Minimum Requirements

- **Node.js**: 20+
- **npm**: 10+
- **OS**: Linux, macOS, Windows (WSL recommended)

---

## Quick Install

```bash
# Clone and install
git clone https://github.com/netdata/ai-agent.git
cd ai-agent
npm install
npm run build

# Or install globally
npm install -g @netdata/ai-agent
```

---

## Verify Installation

```bash
ai-agent --help
```
