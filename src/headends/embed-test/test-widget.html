AI Agent Chat Widget

This page demonstrates the floating chat widget mode. Click the chat button in the bottom-right corner to open the assistant. The widget can be maximized for a full-screen experience.

Persistent

Your conversation history is saved locally and survives page refreshes.

Themeable

Automatically detects your system theme, with manual toggle option.

Markdown

Full markdown support including code blocks and mermaid diagrams.

Streaming

Real-time streaming responses with live status updates.

Embedding the Widget

<!-- Add to your page -->
<link rel="stylesheet" href="https://your-server/test.css">
<script src="https://your-server/ai-agent-public.js"></script>
<script src="https://your-server/test.js"></script>

<!-- Dependencies -->
<script src="https://cdn.jsdelivr.net/npm/markdown-it@14.1.0/dist/markdown-it.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/mermaid@11.12.2/dist/mermaid.min.js"></script>

<!-- Initialize -->
<script>
  new AiAgentChatUI(document.body, {
    mode: 'widget',
    endpoint: 'https://your-server',
    agentId: 'support'
  });
</script>
