Push metrics from your applications to start monitoring.

Quick Start - Push Metrics

curl -X POST http://localhost:19998/api/v1/push \
  -H "Content-Type: application/json" \
  -d '{
    "node_id": "my-app",
    "node_name": "My Application",
    "charts": [{
      "id": "cpu_usage",
      "title": "CPU Usage",
      "units": "%",
      "dimensions": [{
        "id": "user",
        "value": 45.2
      }, {
        "id": "system",
        "value": 12.8
      }]
    }]
  }'

Python Example

import requests
import time
import psutil

while True:
    cpu = psutil.cpu_percent(percpu=True)
    requests.post("http://localhost:19998/api/v1/push", json={
        "node_id": "python-app",
        "charts": [{
            "id": "cpu",
            "title": "CPU per Core",
            "units": "%",
            "dimensions": [
                {"id": f"core{i}", "value": v}
                for i, v in enumerate(cpu)
            ]
        }]
    })
    time.sleep(1)
