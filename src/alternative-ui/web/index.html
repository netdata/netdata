Welcome to Netdata Alternative UI

Push metrics from your applications or Kubernetes clusters to start monitoring.

Kubernetes Monitoring

Monitor pods, nodes, deployments, and cluster resources.

./k8s-collector \
  --url http://localhost:19998 \
  --cluster-name my-cluster

Application Metrics

Push custom metrics from any application.

curl -X POST http://localhost:19998/api/v1/push \
  -H "Content-Type: application/json" \
  -d '{"node_id":"my-app","charts":[...]}'

Kubernetes Collector

# Build the K8s collector
cd k8s && go build -o k8s-collector .

# Run with your cluster
./k8s-collector \
  --url http://localhost:19998 \
  --cluster-name production \
  --namespaces default,kube-system \
  --interval 10s

Python System Metrics

python examples/push_metrics.py \
  --url http://localhost:19998 \
  --node-id my-server \
  --interval 1
